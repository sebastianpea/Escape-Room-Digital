# Escape Room Digital: Mazmorra 🏰门

Un juego de aventura y acertijos desarrollado en **C#** utilizando **Windows Forms**. El objetivo es simple pero desafiante: encontrar la salida de una mazmorra resolviendo enigmas lógicos antes de que sea demasiado tarde.

## 📝 Descripción
Este proyecto es una aplicación de escritorio que simula la experiencia de un "Escape Room". El usuario se encuentra atrapado en una mazmorra y debe interactuar con los elementos visuales de la interfaz para descubrir pistas, descifrar códigos y desbloquear la puerta final.

A diferencia de otros juegos de aventura, este se centra puramente en la resolución de acertijos lógicos y la observación, sin depender de un sistema de inventario.

## 🚀 Características
- **Interfaz Gráfica Interactiva:** Creada con Windows Forms para una experiencia visual clara.
- **Acertijos de Lógica:** Desafíos diseñados para poner a prueba el ingenio del jugador.
- **Navegación por Escenas:** Sistema de cambio de estados para explorar diferentes rincones de la mazmorra.
- **Sin Inventario:** Todo lo necesario para escapar está a la vista o dentro de los acertijos, eliminando la complejidad innecesaria.

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** C#
* **Framework:** .NET 8.0 (Windows Forms)
* **IDE:** Visual Studio 2022
* **Paradigma:** Programación Orientada a Objetos (POO)
